## Install

<Tabs syncKey="install">
  <TabItem label="Composer" icon="seti:php">
    ```bash
    composer require --dev miropen/mir-php
    vendor/bin/mir src/
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install mir-php
    mir src/
    ```
  </TabItem>
  <TabItem label="From source" icon="seti:shell">
    ```bash
    git clone https://github.com/jorgsowa/mir.git
    cd mir && cargo build --release
    ./target/release/mir src/
    ```
  </TabItem>
</Tabs>
